PALM++ Project: alea library

The alea library contains code for Monte Carlo measurements.

Download, Configuring, Building and Installing

are discussed here.

The library will reside in the file libalea.a in the path specifed in the configuration step.

Documentation

  • Observable classes
    • observable.h contains the base class for Monte Carlo measurements.
    • observableset.h is a collection of observables.
    • obsvalue.h contains traits classes for Monte Carlo measurements.
    • simpleobservable.h contains a basic observable class.
    • simpleobseval.h contains evaluator classes for basic observables.
  • Binning strategies
    • nobinning.h does not do any binning.
    • simplebinning.h does rudimentary binning, using as little memory as possible.
    • detailedbinning.h does elaborate binning, including the possibility to save full time series.

Example

The easiest way to learn a new library is to study an example first. We have provided a simple example, showing the most common usage features.

To build the example go to the example/alea subdirectory of PALM++ directory and type 

make examples
The source code and of the examples is well documented and self-explanatory.

To run the example type 

./detailedbinning < detailedbinning.input
