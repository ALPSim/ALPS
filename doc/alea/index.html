ALPS Project: alea library

The alea library contains code for Monte Carlo measurements.

Documentation

  • General header
    • alea.h includes all header files of the alea library.
  • Observable classes
    • alea/observable.h contains the base class for Monte Carlo measurements.
    • alea/observableset.h is a collection of observables.
    • alea/obsvalue.h contains traits classes for Monte Carlo measurements.
    • alea/simpleobservable.h contains a basic observable class.
    • alea/simpleobseval.h contains evaluator classes for basic observables.
  • Binning strategies
    • alea/nobinning.h does not do any binning.
    • alea/simplebinning.h does rudimentary binning, using as little memory as possible.
    • alea/detailedbinning.h does elaborate binning, including the possibility to save full time series.

Example

The easiest way to learn a new library is to study an example first. We have provided a simple example, showing the most common usage features.
