ALPS Project

Header file alps/os.h

This header contains low level operating system call wrappers.

Synopsis

nameespace alps {
std::string hostname();
double dclock();
double dclock_resolution();
}

Functions

std::string hostname();
returns the name of the machine the process is running on. 
double dclock();
returns the wallclock time passed since the first call to dclock() in seconds. This is not synchronized over nodes in a parallel machine. Clock wraparound is treated correctly if the function is called at least once per period. 
double dclock_resolution();
The resolution of the timing returned by dclock()
