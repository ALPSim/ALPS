ALPS Project: xml library

Header file palm.h

This header contains a function to read and write a Parameters object of the palm library into a std::ostream, following the schema presented at http://xml.comp-phys.org/.

Synopsis

nameespace alpsxml {
  void write_xml(std::ostream&, const palm::Parameters&);
  void read_xml(XMLTag, std::istream&, palm::Parameters&);
  void read_xml(std::istream&, palm::Parameters&);
}

Functions

void write_xml(std::ostream&, const palm::Parameters&);
writes the contents of a Parameters object of the palm library into a std::ostream, following the schema presented at http://xml.comp-phys.org/ 
void read_xml(XMLTag tag, std::istream&, palm::Parameters&);
parses a <PARAMETERS> element from an input stream, assuming the schema presented at http://xml.comp-phys.org/. The opening tag is passed as first argument. 
void read_xml(std::istream&, palm::Parameters&);
reads the opening tag of the element and calls above function.
