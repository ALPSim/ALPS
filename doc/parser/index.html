ALPS Project: xml library

The XML library contains simple helper functions for a very primitive XML parser. We aim to replace it with a good C++ XML as soon as one is available.

Download, Configuring, Building and Installing

are discussed here.

The library will reside in the file libxml.a in the path specifed in the configuration step.
