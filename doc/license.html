References to cite in publications

In order to encourage contributions to the library and to be able to obtain support for further development, maintenance and support of the library, the ALPS contributors require that the use of the ALPS library be acknowledged and referenced in any publication based on the library. In particular the following publications shall be referenced:
  • If the SSE application of the ALPS project was used: F. Alet, S. Wessel and M. Troyer, Report cond-mat/03?????.
  • If the classical Monte Carlo program for spin systems is used: M. Troyer and M. Körner, in Bull. Am. Phys. Soc. (2000).
  • For any other Monte Carlo program or if the scheduler library is used: M. Troyer, B. Ammon and E. Heeb, Lec. Notes in Comp. Sci. 1505, 191 (1998).
  • For any other use of the library refer to the ALPS project web page http://alps.comp-phys.org/
