HDF5 Documentation¶

This module contains two classes: iArchive and oArchive, used to read or write HDF5 files

class pyalps.hdf5.iArchive¶

Bases: Boost.Python.instance

an archive class to read HDF5 files

dimensions()¶
dimensions( (iArchive)arg1, (str)arg2) -> int :
returns the number of dimensions of an array at the specified path in the HDF5 file
extent()¶
extent( (iArchive)arg1, (str)arg2) -> object :
returns a list of the extents along any of the dimensions of an array at the specified path in the HDF5 file
filename¶

the (read-only) file name of the archive

is_attribute()¶
is_attribute( (iArchive)arg1, (str)arg2) -> bool :
returns True if the given path points to an attribute in the HDF5 file
is_data()¶
is_data( (iArchive)arg1, (str)arg2) -> bool :
returns True if the given path points to data in the HDF5 file
is_group()¶
is_group( (iArchive)arg1, (str)arg2) -> bool :
returns True if the given path is a group in the HDF5 file
is_null()¶
is_null( (iArchive)arg1, (str)arg2) -> bool :
returns True if the given path points to an empty (null) node in the HDF5 file
is_scalar()¶
is_scalar( (iArchive)arg1, (str)arg2) -> bool :
returns True if the given path points to scalar data in the HDF5 file
list_attr()¶
list_attr( (iArchive)arg1, (str)arg2) -> list :
returns a list with the names of all the attributes of a given path in the HDF5 file
list_children()¶
list_children( (iArchive)arg1, (str)arg2) -> list :
returns a list with the relative paths of all child nodes relative to a given path in the HDF5 file
read()¶
read( (iArchive)arg1, (str)arg2) -> object :
reads an object from the specified path in the HDF5 file. Currently supported types are scalar types and numpy arrays of them.
