Alea Documentation¶

This module contains classes for the aggregation of Monte Carlo measurements:

  • RealObservable
  • RealVectorObservable
  • RealTimeSeriesObservable
  • RealVectorTimeSeriesObservable

and two classes for the evaluation of Monte Carlo measurements:

  • MCScalarData
  • MCVectorData

This module contains classes for the evaluation of Monte Carlo measurements: - RealObservable - RealVectorObservable - RealTimeSeriesObservable - RealVectorTimeSeriesObservable

class pyalps.alea.MCScalarData¶

This class is used to evaluate Monte Carlo data and functions on them. Besides the documented functions and properties, the class supports the arithmetic operations +, -, *, /, +=, -=, *=, /=, and the functions abs, acos, acosh, asin, asinh, atan, atanh, cb, cbrt, cos, cosh, exp, log, pow, sin, sinh, sq, sqrt, tan and tanh.
