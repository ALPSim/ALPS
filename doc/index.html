The PALM++ Project

The PALM++ project (Parallel Algorithms for Lattice Models) aims at providing generic parallel algorithms for classical and quantum lattice models and provides utility classes and algorithm for many other problems. It strives to increase software reuse in the physics community.

Libraries

Currently the following libraries are publicly available as part of the PALM++ library project

  • palm generally usefull classes and traits
  • osiris serialization and communication
  • alea Monte Carlo measurements and evaluation
  • xml helper functions to parse and write XML files
  • lattice lattices and graphs
  • scheduler an automatically parallelizing Monte Carlo library

Tools

Besides the helper tools for the scheduler library we have provided a few sample tools for the manipulation of the XML files created by the PALM++ programs.

Programs

are not yet available publicly, but you can contact us if you are interested. Current projects include
  • Classical Monte Carlo simulations
    • Generic local update and cluster Monte Carlo algorithms for classical lattice models
  • Quantum Monte Carlo simulations
    • Loop algorithm
    • Worm algorithm
    • Stochastic Series Expansion
  • Diagonalization of quantum lattice models
    • Exact diagonalization using expression templates
    • Full diagonalization using expression templates
    • Parallel exact diagonalization
  • Series expansions on quantum lattice models
    • High temperature series using expression templates
  • Mean field calculations on quantum lattice models using expression templates
