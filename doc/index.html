The ALPS Project

The ALPS project (Parallel Algorithms for Lattice Models) aims at providing generic parallel algorithms for classical and quantum lattice models and provides utility classes and algorithm for many other problems. It strives to increase software reuse in the physics community.

Libraries

Currently the following libraries are publicly available as part of the ALPS library project

  • alps generally usefull classes and traits
  • alps/osiris serialization and communication
  • alps/alea Monte Carlo measurements and evaluation
  • alps/parser helper functions to parse and write XML files
  • alps/lattice lattices and graphs
  • alps/model quantum lattice models
  • alps/scheduler an automatically parallelizing scheduler for Monte Carlo simulations

Tools

Besides the helper tools for the scheduler library we have provided a few sample tools for the manipulation of the XML files created by the ALPS programs.

Programs

are not yet available publicly, but you can contact us if you are interested. Current projects include
  • Classical Monte Carlo simulations
    • Generic local update and cluster Monte Carlo algorithms for classical lattice models (beta-release avaliable)
  • Quantum Monte Carlo simulations
    • Loop algorithm (beta-release avaliable)
    • Directed loop algorithm (beta-release avaliable)
    • Worm algorithm (beta-release avaliable)
  • Diagonalization of quantum lattice models
    • Exact diagonalization using expression templates
    • Full diagonalization using expression templates
    • Parallel exact diagonalization
  • Series expansions on quantum lattice models
    • High temperature series
    • Perturbation series
